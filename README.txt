ABOUT

Practice code, implementing the rules of the Goat, Wolf, & Cabbage social deduction game, Secret Hitler. Information about the game add rules is available at secrethitler.com.

USAGE

Product is currently viable. May be run from command line:

shMain.py
  arguments:
    -debug -- turns on logging messages for debugging
    -test -- will auto-generate a list of players for testing purposes

or

Main.class (Java Implementation)
